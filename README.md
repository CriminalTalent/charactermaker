# OC 캐릭터 키워드 시트 사용 가이드

---

## 기본 개념

이 툴은 해리포터 세계관 기반의 자캐(OC)를 만들 때, 단순히 설정을 메모하는 게 아니라 **"외형 → 키워드 → 심리 → 관계 → 요약"** 흐름으로 캐릭터를 입체적으로 설계할 수 있도록 만들어졌어요.

가장 중요한 개념은 **전역 자캐 선택**이에요. 헤더 아래 바에서 자캐를 한 번 클릭하면, 4·5·6탭에 그 자캐 데이터가 자동으로 연결돼요.

---

## 탭별 설명

### ① 캐릭터 입력폼
자캐 목록이 보이는 메인 화면이에요.

**올랜덤 생성** 버튼을 누르면 외형부터 비밀 설정까지 즉시 생성돼요. 마음에 들면 저장, 아니면 다시 굴리기.

**일부 고정** 버튼을 누르면 성별·기숙사·이름 스타일 등을 미리 지정하고 나머지만 랜덤으로 돌릴 수 있어요. 이름은 한국식·일본식·서양식 중에서 고를 수 있고, 기본은 랜덤이에요. 이름 표기는 항상 **한글(영문)** 형식으로 나와요.

**+ 직접 입력**은 모달 창으로 6단계(외형 → 기본 설정 → 마법 설정 → 성격 키워드 → 기숙사 → 비밀 설정)를 직접 채우는 방식이에요.

카드에 있는 **⧉ 버튼**은 복제, **✕ 버튼**은 삭제예요.

---

### ② 키워드 사전
캐릭터 설계에 쓸 수 있는 **80개 이상의 내장 키워드**가 분류별로 정리돼 있어요.

대분류(분위기·성격·심리·관계·서사·상징 등)로 탭 필터를 걸거나, 검색으로 원하는 키워드를 찾을 수 있어요. **톤 필터**(서정적·차가운·무거운 등)와 **강도 필터**(낮음·보통·높음)도 함께 쓸 수 있어요.

강도의 의미는 이래요.
- **낮음** — 배경에 은은하게 깔리는 요소
- **보통** — 상황에 따라 드러났다 숨었다 하는 요소
- **높음** — 캐릭터 핵심에 강하게 박혀 있어서 숨기기 어려운 요소

각 카드에는 유사 키워드·반대 키워드·잘 붙는 키워드·위험 조합이 적혀 있어요. 위험 조합(빨간 배지)은 이 키워드를 쓸 때 빠지기 쉬운 클리셰예요.

**+ 키워드 추가**로 나만의 키워드를 직접 추가할 수도 있어요. 직접 추가한 키워드는 파란 배지로 구분돼요.

---

### ③ 조합 규칙표
키워드를 **최대 3개 선택**하고 조합 분석하기를 누르면 아래 항목이 나와요.

- **조합 인상** — 시너지 수준(높음·보통·낮음)과 지배적인 톤
- **시너지 연결** — 서로 pairs에 등록된 쌍이 있으면 표시
- **클리셰 위험도** — 높음 강도 키워드가 많을수록 올라가는 바 그래프
- **비틀기 제안** — 선택된 키워드를 진부하지 않게 비틀 수 있는 아이디어
- **충돌 요소** — 반대 방향이거나 위험 조합인 쌍 경고
- **다음에 붙이면 좋은 키워드** 추천

---

### ④ 심리/서사 축
캐릭터의 내면 구조를 설계하는 탭이에요. 크게 두 파트예요.

**왼쪽 — 심리 구조**
욕망·결핍·두려움·수치심·방어기제·분노 트리거·무너지는 상황·애정 표현 방식, 그리고 스스로 믿는 자기서사 vs 실제 본질을 입력해요. 각 항목 옆 **주사위 버튼**을 누르면 랜덤으로 뽑아줘요.

**오른쪽 — 서사 구조**
성장 방향과 파멸 방향을 입력하면 하단에 **초록/빨강 분기 시각화**가 생겨요.

**심리 번들 라이브러리** 8종(겨울 아이·완벽주의자·헌신하는 자·야심가·관찰자·생존자·방랑자·분노한 자)을 클릭하면 관련 항목이 한 번에 채워져요. 출발점으로 쓰고 이후에 수정하는 방식으로 쓰면 좋아요.

헤더 바에서 자캐를 선택하면 1탭에서 입력한 심리 관련 데이터가 자동으로 불러와져요.

---

### ⑤ 관계/운용
자캐가 어떻게 굴러가는지를 설계하는 탭이에요.

**관계 유형** 10종(관찰자형·헌신형·집착형·경쟁형·순환 이탈형 등)을 선택하고, 잘 맞는 상대·안 맞는 상대·가까워지는 계기·멀어지는 계기·신뢰/배신 트리거·감정 트리거(연애·질투·보호본능·열등감)를 입력해요. 모든 항목에 주사위 버튼이 있어요.

**관계 번들** 6종도 있어요.

**자캐 간 관계 메모** 섹션에서는 저장된 자캐 두 명을 A·B로 선택해서 관계 유형·갈등 포인트·한줄 설명을 메모할 수 있어요. 같은 쌍을 다시 선택하면 기존 내용이 불러와져요.

---

### ⑥ 최종 요약 카드
1~5탭에 흩어진 데이터를 한 장으로 모아주는 탭이에요.

헤더 바에서 자캐를 선택하거나 이 탭 안의 자캐 선택 버튼을 누르면 자동으로 카드가 완성돼요. 외형·성격 키워드·심리 구조·관계 성향·마법 설정·핵심 모순·가장 아픈 질문·성장/파멸 방향이 한 번에 보여요.

**직접 입력 모드**를 펼치면 자캐 저장 없이 즉석에서 카드를 만들 수도 있어요.

**인쇄/PDF** 버튼을 누르면 헤더와 네비게이션이 사라지고 카드만 깔끔하게 출력돼요.

---

## 데이터 관리

모든 데이터는 **브라우저 localStorage**에 자동 저장돼요. 새로고침해도 사라지지 않아요.

다만 브라우저 캐시를 지우거나 다른 기기에서는 보이지 않기 때문에, 중요한 자캐는 헤더의 **내보내기**로 JSON 파일을 백업해두는 걸 권장해요. 나중에 **불러오기**로 다시 복원할 수 있어요.

---

## 추천 사용 순서

처음 자캐를 만들 때는 이 흐름이 편해요.

1. **올랜덤 생성**으로 초안을 뽑고 저장
2. 헤더 바에서 그 자캐 선택
3. **② 키워드 사전**에서 캐릭터에 맞는 키워드 찾고
4. **③ 조합 규칙표**에서 키워드 조합 점검
5. **④ 심리 축**에서 번들 하나 골라서 내면 구조 정리
6. **⑤ 관계 탭**에서 관계 패턴 정리
7. **⑥ 최종 카드**에서 완성본 확인 후 X 공유 또는 PDF 저장
